# 文章与上传模块

最后更新：2026-06-05

## 一、模块范围

文章与上传模块负责：

- 文章分类。
- 文章列表。
- 文章详情。
- 文章新增、编辑、删除。
- 草稿、发布、下架。
- 文章封面。
- 文章附件。
- 本地文件上传。
- wangEditor 富文本图片上传。
- `/api/common/files/**` 受控文件访问。

评论归档到 `7.评论与敏感词模块.md`。

## 二、数据模型

| 表 | 说明 |
| :--- | :--- |
| `article_category` | 文章分类 |
| `article` | 文章主表 |
| `article_attachment` | 文章附件关系 |
| `file_resource` | 上传文件元数据 |

文章核心字段：

- 标题。
- 分类。
- 封面 URL。
- 摘要。
- 富文本 HTML。
- 纯文本内容。
- 可见性。
- 状态：草稿、已发布、下架。
- 浏览数。
- 评论数。
- 发布时间。

## 三、接口

| 接口 | 方法 | 说明 |
| :--- | :--- | :--- |
| `/api/article/types` | GET | 文章分类 |
| `/api/article/list` | GET | 文章分页 |
| `/api/article/detail/{id}` | GET | 文章详情 |
| `/api/article` | POST | 新增文章 |
| `/api/article/{id}` | PUT | 编辑文章 |
| `/api/article/{id}` | DELETE | 删除文章 |
| `/api/article/{id}/status` | PATCH | 修改文章状态 |
| `/api/article/upload` | POST | 文章相关上传 |
| `/api/common/files/**` | GET | 文件读取 |

## 四、权限码

## 五、业务规则

文章：

- 标题不能为空。
- 分类必须存在且启用。
- 正文不能为空。
- 富文本保存前净化。
- 发布状态需要维护发布时间。
- 删除使用逻辑删除。

上传：

- 校验文件后缀。
- 校验 MIME。
- 校验大小。
- Spring multipart 单文件和单请求上限已配置为 50MB，避免业务层 50MB 限制被框架默认 1MB 限制提前拦截。
- 超出 multipart 限制时返回统一 400 响应，提示“上传文件大小不能超过50MB”。
- Office 文件在后缀白名单命中时，兼容部分浏览器上传的 `application/octet-stream`、`application/zip`、`application/x-zip-compressed` MIME。
- SVG 暂不开放上传，原因是 SVG 可包含脚本、外链和事件属性；后续如需支持必须先增加 SVG 净化和安全渲染策略。
- 生成后端文件名。
- 规范化存储路径，防止目录穿越。
- 文件读取必须命中 `file_resource` 元数据。

附件：

- 保存文章时校验附件 ID 必须存在且未删除。
- 普通用户只能引用本人上传的附件；超级管理员保留管理例外。
- 保存后重写文章附件关系。
- 后续如引入多组织内容协作，再扩展组织或数据权限级附件引用规则。
