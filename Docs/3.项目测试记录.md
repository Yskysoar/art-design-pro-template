# 项目测试记录

最后更新：2026-06-03

## 一、测试策略

测试按层级拆分：

| 层级 | 工具 | 目的 |
| :--- | :--- | :--- |
| Service 单元测试 | JUnit 5 + Mockito + AssertJ | 验证业务规则、边界条件和异常分支。 |
| Controller 测试 | Spring Boot Test + MockMvc + spring-security-test | 验证认证、权限、参数校验和统一响应。 |
| 数据库回归 | MySQL + schema/mock SQL | 验证表结构、Mock 数据和真实接口。 |
| 前端类型检查 | `vue-tsc --noEmit` | 验证 Vue + TypeScript 类型正确。 |
| 前端构建 | `vite build` | 验证生产构建。 |
| 浏览器回归 | Codex 内置浏览器或手工检查 | 验证页面跳转、布局、交互和接口联调。 |


数据库重置注意事项：

- 只能操作 `backend-template` 数据库。
- 导入 UTF-8 SQL 时使用 `cmd /c` 原始重定向，并指定 `--default-character-set=utf8mb4`。
- 不使用 PowerShell `Get-Content | mysql` 管道导入中文 SQL。

## 四、模块测试用例

### 4.1 认证与账号

已覆盖或应覆盖：

- 登录成功。
- 登录验证码错误失败。
- 注册成功。
- 注册用户名重复失败。
- 注册验证码错误失败。
- 重置密码成功。
- 重置密码验证码错误失败。
- 当前用户修改密码旧密码错误失败。
- 当前用户修改密码成功后前端要求重新登录。
- 无效或过期 JWT 返回 401。
- 公开验证码接口不被 JWT 过滤器拦截。

### 4.2 系统管理

已覆盖或应覆盖：

- 用户列表 401/403/200。
- 用户新增、编辑、启停用、删除。
- 用户角色关系和组织关系保存。
- 角色列表、新增、编辑、删除。
- 角色菜单权限和数据权限保存。
- 菜单管理树、新增、编辑、删除。
- 组织树数据权限过滤。
- 配置项唯一键、只读保护、逻辑删除后同名重建。
- 敏感词库分页、新增、编辑、启停用、删除。

### 4.3 文章与上传

已覆盖或应覆盖：

- 文章分类只返回启用分类。
- 文章列表分页和筛选。
- 文章新增、编辑、状态变更和删除权限。
- 富文本保存前净化。
- 附件存在性校验。
- 上传后缀、MIME、大小和路径穿越。
- 文件读取必须命中落库资源。
- 文章详情隐藏路由浏览器刷新。

### 4.4 评论与敏感词

已覆盖或应覆盖：

- 评论列表公开读取跟随文章可访问性。
- 游客不能读取草稿、下架或隐藏文章评论。
- 登录用户新增一级评论和回复。
- 空内容、超长内容失败。
- 跨文章回复失败。
- 已删除评论不可回复。
- 隐藏和删除评论内容脱敏。
- 评论数量原子增减。
- 公开响应不返回内部 `userId`。
- 评论发布命中敏感词失败，返回命中词。
- 敏感词表缺失时评论发布路径降级，不直接 500。

### 4.5 社交关注与聊天

已覆盖或应覆盖：

- 不能关注自己。
- 重复关注幂等成功，不重复插入。
- 取消关注只删除单向关系。
- 拉黑后自动取消双方关注。
- 拉黑后不能关注和私信。
- 非互关连续 3 条私信成功，第 4 条失败。
- 对方回复后额度恢复 3 条。
- 互关后私信不限。
- 取消互关后恢复额度规则。
- 文本为空或超长发送失败。
- 私信命中敏感词失败。
- 会话列表返回最后消息和未读数。
- 打开会话后对方消息标记已读。

当前已新增 `SocialServiceImplTest`，已覆盖不能关注自己、拉黑取消双方关注、拉黑后不能关注、非互关第 4 条私信失败、互关私信不限额、私信敏感词拦截、会话标记已读。重复关注、取消关注、对方回复后额度恢复、会话列表和浏览器交互需在后续联调中继续补充。
